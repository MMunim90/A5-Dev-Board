Question-1: What are the different ways to select an element in the DOM?


There are 5 ways to select an element in the DOM. They are:-

1.Using getElementById. | Ex: const element = document.getElementById("idName");


2.Using getElementsByClassName. | Ex: const elements = document.getElementsByClassName("className");


3.Using getElementsByTagName. | Ex: const elements = document.getElementsByTagName("h1");


4.Using querySelector. | Ex: const element = document.querySelector("#idName");


5.Using querySelectorAll. | Ex: const elements = document.querySelectorAll(".className");

Question-3: What is event delegation in the DOM?


Event delegation is a technique in JavaScript, where adding a single event listener
to parent element instead of adding event listeners to multiple child elements.


Question-4: What is event bubbling in the DOM?


Event bubbling in JavaScript is a mechanism where an event triggered on a child element
propagates upward through its ancestors in the DOM. It allows parent elements to respond
to events triggered by their child elements.


Question-5: How do you create, add, and remove elements using JavaScript?


1. Create Elements using JavaScript:-
=> To create an element using JavaScript, we use .createElement("p")
Ex: const paragraph = document.createElement("p");

2. Add Elements using JavaScript:-
=> To add an element using JavaScript, we use .appendChild()
Ex: const element = document.getElementById("div1");
element.appendChild(paragraph);

3. Remove Elements using JavaScript:-
=> To remove an element using JavaScript, we use remove()
Ex: document.getElementById("div1").remove();
